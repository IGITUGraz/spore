Detailed Description

template<typename targetidentifierT>
class spore::SynapticSamplingRewardGradientConnection< targetidentifierT >

Reward-based synaptic sampling connection class.

SynapticSamplingRewardgradientConnection - Synapse implementing reward-based learning through synaptic sampling. This class implements the model in [1,2]. This connection is a diligent synapse model, therefore updates are triggered on a regular interval which is ensured by the ConnectionUpdateManager. The target node and the reward transmitter must be derived from the TracingNode model.

direct_gradient_rate double rate of directly applying changes to the synaptic parameter
gradient_scale double scaling parameter for tde gradient
reward_transmitter int GID of tde synapse's reward transmitter
bap_trace_id int ID of tde BAP trace (default 0)
dopa_trace_id int ID of tde dopamine trace (default 0)
simulate_retracted_synapses bool continue simulating retracted synapses (default false)
delete_retracted_synapses bool delete retracted synapses. simulate_retracted_synapses must also be true for this to be effective. (default false)
verbose bool write status to std::out (for debugging, default false)

reset_recorder bool clear all recorded values now (write only)

Send an event to the postsynaptic neuron. This will update the synapse state and synaptic weights to the current slice origin and send the spike event. This method is also triggered by the ConnectionUpdateManager to indicate that the synapse is running out of data. In this case an invalid rport of -1 is passed and the spike is not delivered to the postsynaptic neuron.
